# Audio Files

Add these free audio files before deploying:

## Music tracks (MP3)
- `title.mp3` — lofi/chiptune love theme (title screen)
- `level.mp3` — upbeat chiptune platformer track (playable levels)
- `cutscene.mp3` — slow, gentle piano (cutscenes)
- `cliffhanger.mp3` — mysterious, building tension (chapter 8)

## SFX (MP3)
- `jump.mp3` — jump sound
- `collect.mp3` — coin/item collect
- `complete.mp3` — level complete fanfare

## Free sources
- Kenney Audio packs (CC0): https://kenney.nl/assets/category:Audio
  - Use: Impact Sounds, UI Audio, Sci-Fi Sounds packs
- freesound.org (filter by CC0)
- opengameart.org

The game works without audio — missing files are silently skipped.
